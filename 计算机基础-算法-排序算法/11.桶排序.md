# 桶排序

- 用于位数相同的小数排序,范围=[0,1)

## 核心思想:

- 分组,各个组内排序

```
826	211	284	502	210	335	264
```

## 基本流程:

- 找到最小值和最大值并根据最高位进行分组

  - Min:	210 
  - Max:	826
  - 得到分组[2,8],应用于数组中:[0,6]

- 将元素入桶

  | 0    | 1    | 2    | 3    | 4    | 5    | 6    |
  | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
  | 211  | 335  |      | 502  |      |      | 826  |
  | 284  |      |      |      |      |      |      |
  | 210  |      |      |      |      |      |      |
  | 264  |      |      |      |      |      |      |

- 之后仅需要对桶内元素进行排序即可

  - 这里要使用链表排序
  - 可使用
    - 冒泡排序
    - 选择排序
    - 插入排序

## 代码思路:

1. 寻找最高/最低位
2. 确定位数
3. 按照高位进行桶的申请
4. 元素入桶
5. 桶内排序
6. 放回原数组
7. 释放桶
